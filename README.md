# TodoApp

Решение тестового задания REST API для управления списком задач.

## Запуск проекта

При запуске открывается scalar http://localhost:5105/scalar/v1#tag/todoapp

## Технологический стек

- `Сервер` - ASP.NET Core WebAPI;
- `OРМ` - EF Core;
- `База данных` - In-Memory;
- `Маппинг` - Mapster;
- `Язык` - C#
